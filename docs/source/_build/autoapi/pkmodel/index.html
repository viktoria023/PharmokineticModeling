pkmodel¶

pkmodel is a Pharmokinetic modelling library.

It contains functionality for creating, solving, and visualising the solution of Parmokinetic (PK) models

class pkmodel.Model(value, Q_p1=1.0, V_c=1.0, V_p1=1.0, CL=1.0, X=1.0, k_a=None, stop=None, scheme=None, tps=None)¶

A Pharmokinetic (PK) model

value: numeric, an example parameter to test if class can be called correctly Q_p1: [mL/h] float, the transition rate between central compartment and peripheral compartment 1 V_c: [mL] float, the volume of the central compartment V_p1: [mL] float, the volume of the first peripheral compartment CL: [mL/h] float, the clearance/elimination rate from the central compartment X: [ng] float, the amount of drug administered per injection k_a: [/h] float is the absorption rate, only used for the sc dosing scheme: [‘clt’,’hlt’,’dt’] pattern of dose administration as specified by user. clt = instantaneous limited time, hlt = hourly limited time, dt = defined time points stop: [h] float, timepoint at which injection stops. Only used in ‘ilt’ and ‘hlt’ scheme tps: [h] float, timepoints of dose administration as specified by the user. Only used in ‘dt’ scheme

Setting up intravenous model Sets up 2-compartment model
